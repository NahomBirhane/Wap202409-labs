# CS472 - Homework 01

## Task 1: Identify Block vs. Inline HTML Elements

### Block Elements:
- `<div>`
- `<p>`
- `<h1>` to `<h6>`
- `<hr>`
- `<ol>` / `<li>`
- `<ul>` / `<li>`
- `<table>` / `<tr>` / `<th>` / `<td>`
- `<form>`

### Inline Elements:
- `<span>`
- `<br>`
- `<img>`
- `<a>`
- `<input>`
- `<textarea>`
- `<button>`
- `<select>` / `<option>`
- `<label>`
